# Archived Packages

This directory contains packages that have been deprecated and are no longer actively maintained or published. The code is preserved here for reference.

## `drizzle/`

**`@react-querybuilder/drizzle`** — Deprecated in favor of the built-in `"drizzle"` export format in `@react-querybuilder/core`. Use `formatQuery(query, 'drizzle')` instead. See the [documentation](https://react-querybuilder.js.org/docs/utils/export#drizzle-orm) for details.

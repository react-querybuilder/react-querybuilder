## React Query Builder Rules Engine Example

[![Open in CodeSandbox](https://img.shields.io/badge/Open_in-CodeSandbox-000000?logo=codesandbox)](https://react-querybuilder.js.org/sandbox?t=rules-engine) [![Open in StackBlitz](https://img.shields.io/badge/Open_in-StackBlitz-1269D3?logo=stackblitz)](https://react-querybuilder.js.org/sandbox?p=sb&t=rules-engine)

Click one of the links above to open this example as an online, editable demo, or clone it locally with [degit](https://www.npmjs.com/package/degit):

```bash
npx degit github:react-querybuilder/react-querybuilder/examples/rules-engine rqb-example-rules-engine
```

This example builds an if/then/else rules engine with [`@react-querybuilder/rules-engine`](https://www.npmjs.com/package/@react-querybuilder/rules-engine), exports it to [json-rules-engine](https://github.com/CacheControl/json-rules-engine) and [rulepilot](https://github.com/andrewbrg/rulepilot) formats, and runs it against a set of sample facts.
